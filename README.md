# Dhruva Hegde — Portfolio Website

A professional, responsive portfolio website built with pure **HTML, CSS, and JavaScript** — no frameworks, no dependencies, just clean code.

## 🌐 Live Demo : https://dhruvahegde.github.io/FUTURE_FS_01/

## 🌐 Live Preview

Open `index.html` directly in your browser, or serve it locally:

```bash
# Using VS Code Live Server extension (recommended)
# Right-click index.html → Open with Live Server

# Or using Python
python -m http.server 8080
# Visit: http://localhost:8080
```

## 📁 Project Structure

```
portfolio/
├── index.html      # Full HTML structure (all 6 sections)
├── style.css       # Dark theme, animations, responsive layout
├── script.js       # Typewriter, scroll-reveal, form, interactions
└── README.md       # This file
```

## ✨ Features

- **Hero Section** – Typewriter animation, floating badges, gradient avatar
- **About Section** – Bio, education details, animated stat cards
- **Skills Section** – 8 skill cards with custom icon colors and hover effects
- **Achievements Section** – Hackathon, NPTEL cert, Internship cards
- **Projects Section** – Project cards with tilt effect on hover
- **Contact Section** – Client-side validated form + full contact details
- Smooth scroll-reveal animations (Intersection Observer API)
- Sticky glassmorphism navbar with active section tracking
- Fully responsive (mobile, tablet, desktop)
- SEO meta tags + Open Graph tags

## 🛠️ Tech Stack

| Layer     | Technology          |
|-----------|---------------------|
| Markup    | HTML5 (Semantic)    |
| Styling   | Vanilla CSS (Custom Properties, Grid, Flexbox) |
| Logic     | Vanilla JavaScript (ES6+) |
| Icons     | Font Awesome 6      |
| Fonts     | Google Fonts (Inter, JetBrains Mono) |

## 🚀 Deployment Options

### GitHub Pages (Free)
1. Push this folder to a public GitHub repo
2. Go to **Settings → Pages → Branch: main → / (root)**
3. Your site will be live at `https://DhruvaHegde.github.io/portfolio`

### Netlify (Free + Custom Domain)
1. Drag and drop the `portfolio/` folder at [netlify.com/drop](https://app.netlify.com/drop)
2. Get an instant live link

### Vercel
```bash
npx vercel --prod
```
